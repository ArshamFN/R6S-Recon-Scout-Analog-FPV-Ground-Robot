## Session 000 — 2026-03-15: Project Kickoff, Parts Acquired & First Wheel Print

the GitHub repository, and begin physical prototyping with the first 3D-printed
wheel design.

**Summary:** All hardware decisions finalized around a pure analog FPV drone

bidirectional mode, FlySky FS-iA6B receiver, and AKK BA3 AIO FPV camera. Repository
created and documentation structure set up. First PLA wheel prototype printed.
ESCs arrived significantly smaller than anticipated, requiring a full body redesign.
